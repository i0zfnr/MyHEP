Suggested Next Steps For New Agent
----------------------------------
1. Run:
   git status --short --branch
   git log --oneline --decorate -5

2. Inspect remote commits before pulling:
   git fetch origin
   git log --oneline HEAD..origin/main

3. Fix local database availability:
   - Start MySQL/MariaDB in Laragon.
   - Confirm .env DB_DATABASE, DB_USERNAME, DB_PASSWORD, DB_HOST, DB_PORT.
   - Then run:
     php artisan migrate:status

4. Fix frontend build:

Notes For Future Development
----------------------------
- Be careful with routes/web.php. It is the main behavior file and is easy to break because many workflows are inside it.
- Do not assume migrations fully recreate the current database content; the root e-biasiswa.sql dump may contain important schema/data.
- File upload routes use the public disk. Make sure storage link exists:
  php artisan storage:link
- Some roles/scopes are enforced through middleware such as:
  auth.session:student
  auth.session:admin
  admin.scope:system
  admin.scope:scholarship
  admin.scope:discipline
- Helper functions such as auditLog(), downloadCsv(), and option helpers may live in app/Support/helpers.php.
